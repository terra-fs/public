# Terra Public Assets

This repository contains **public, non-sensitive assets** intended to be accessed by third-party platforms such as Slack, email clients, and documentation tools.

## Scope
- Brand assets required to be publicly accessible
- Stable URLs intended for long-term use

## Out of Scope
- Application code
- Credentials or secrets
- Client data or information
- Internal documentation
- Temporary or experimental files

If an asset is here, it is assumed safe for public access.
